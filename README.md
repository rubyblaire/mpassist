# MakePlace Assistant

**MakePlace Assistant** is a Dalamud plugin for **Final Fantasy XIV** that helps turn large MakePlace shopping lists into a cleaner, more manageable marketboard workflow.

Built as a **manual shopping companion** rather than an autobuyer, it helps you stay organized, track your progress, and move through furnishing runs with far less friction.

---

## Features

- Load **MakePlace-exported shopping lists** from a `.txt` file
- Browse for list files directly in-game
- Track **required** and **owned** quantities
- Support for **variant-aware entries**, including dyed items
- Mark items complete as you shop
- Refresh progress during long marketboard sessions
- Clean UI designed for practical, high-volume furnishing runs
- A visual: https://i.imgur.com/CVtT1u5.png
---

## What It Does

MakePlace Assistant helps you work through your furnishing list one item at a time without needing to manually track every purchase yourself.

It is designed to make shopping easier by helping you:

- keep your list visible in-game
- track what you still need
- track what you already own
- stay organized during long shopping sessions
- reduce mistakes when shopping for large builds

---

## What It Does Not Do

MakePlace Assistant is **not** an autobuyer.

It does **not** automatically purchase items from the marketboard. It is intended to assist with manual shopping and list management.

## Installation

### Public / Custom Repo
Install https://raw.githubusercontent.com/rubyblaire/mpassist/main/pluginmaster.json in your experimental.

Search for MakePlace Assistant, install.

---

## Usage

1. Open **MakePlace Assistant** in-game.
2. Load a MakePlace shopping list using the **Browse** button.
3. Review the items and quantities required.
4. Shop manually at the marketboard.
5. Mark items complete and refresh progress as needed while shopping.

---

## Supported File Type

MakePlace Assistant currently expects a **MakePlace shopping list exported as a `.txt` file**.

---

## Commands

- `/mpassist` — Open the plugin window

---

## Known Limitations

- Shopping lists must be loaded from a supported `.txt` export
- Purchasing is still manual
- Item resolution may depend on how a MakePlace list formats certain entries or variants

---

## Why This Plugin Exists

Shopping for large MakePlace builds can be one of the most tedious parts of the decorating process.  
MakePlace Assistant was created to make that process smoother, more readable, and less mentally exhausting.
